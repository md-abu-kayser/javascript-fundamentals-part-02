# JavaScript Conditionals Mastery

A polished learning and interview-prep repository covering JavaScript conditional logic, branching, nested conditions, logical operators, and ternary expressions.

## 🚀 Project Overview

This repository teaches and documents core JavaScript conditional concepts through explanatory examples, practical tasks, and working solutions.

It is designed for:

- beginners learning JavaScript control flow
- developers preparing for technical interviews
- educators sharing fundamental conditional patterns

## 📁 Repository Structure

- `js-fundamental-conditionals/` — core examples and walkthroughs for conditionals, comparisons, logical operators, nesting, and ternary usage.
- `js-conditional-tasks/` — exercise implementations for common conditional problems.
- `js-conditional-tasks-solution/` — reference solutions to the same set of tasks.
- `JS_CONDITIONALS.md` — conceptual notes, operator explanations, and practical examples.

## ✅ What You Will Learn

- Comparison operators: `==`, `===`, `!=`, `!==`, `>`, `<`, `>=`, `<=`
- Basic `if` and `if...else` statements
- Multi-branch logic with `else if`
- Nested conditional structures
- Boolean logic using `&&`, `||`, `!`
- Ternary operator shorthand
- Conditional problem solving with real task examples

## 💡 Key Highlights

- Practical code examples built with plain JavaScript.
- Clear separation between learning examples and task solutions.
- Interview-ready explanations and best practices.
- Clean, modular files that make it easy to explore each concept.

## ▶️ How to Run the Examples

Use Node.js to execute any script from the command line.

```bash
cd c:\projects\javascript-fundamentals-part-02
node js-fundamental-conditionals/if.js
node js-conditional-tasks/1_Free_Drinks.js
```

## 🧠 Tasks Included

The exercise set includes common interview-style conditional challenges such as:

1. Free drinks eligibility
2. BMI calculator
3. Grade calculator
4. Nested friend check
5. Ternary-based number decision
6. Bus fare calculator

## 📘 Suggested Learning Path

1. Review `JS_CONDITIONALS.md` for a conceptual foundation.
2. Explore `js-fundamental-conditionals/` to see core patterns.
3. Practice with `js-conditional-tasks/` exercises.
4. Compare your implementation to `js-conditional-tasks-solution/`.

## 🎯 Interview Preparation

This repository is structured to help you answer questions like:

- "What is the difference between `==` and `===`?"
- "How do you write nested `if` statements?"
- "When should you use a ternary operator?"
- "How do logical operators affect conditional evaluation?"
